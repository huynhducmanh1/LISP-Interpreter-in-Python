The program need to run with Python 3 and all commands need to be capitalized
There is a prompt when run REPLtest.py to begin the program (1 for read test files/ 2 for interactive mode)
There is two test files: test.txt is for all commands testing (formar: [command: output]), fizzbuzz.txt output a FIZZBUZZ program

List of python files:
  REPLtest.py: the main read-evaluation-print loop. (Also option for reading tests file)
  typestest.py: define data types for LISP (S-expression, Number expression, Symbol expression, List)
  reader2.py: lexer and parser, return with approriate data types. 
  printer2.py: print according to the result/data types
